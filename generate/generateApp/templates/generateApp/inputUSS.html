User Story Scenario

Scenario:
Given:

The user is a student who has logged into the attendance system.

The user has recorded attendance history for several courses.

When:

The user navigates to the "Attendance History" page.

Then:

The system displays the attendance records for the selected course and time period.

When:

The student selects the desired course and date range and presses the "Show History" button.

Then:

The system displays the student's attendance history in a table with date, status, and total percentage.
